#Installation & setup

1. Clone the repository
  
    ```bash
     git clone https://github.com/bishakhne0gi/DashboardUI.git
     ```
2. Change the directory
    ```bash
    cd DashboardUI
    ```

2. Install the node modules

    ```bash
    npm install
    ```
3. Run the server

    ```bash
    npm start
    ```
